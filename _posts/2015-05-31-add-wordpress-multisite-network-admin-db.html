Recently, I needed to add a Network Admin (Super Admin) to the WordPress Multisite via Database and here is a quick article to show you how I did it with phpMyAdmin.

In this example, we add an existing user - Josh as a new Network Admin.

  1. Go to table wp_sitemeta
  2. Search for a meta_key called site_admins
  3. The value will look like the following syntax:
    a:3:{i:0;s:5:"admin";i:1;s:3:"tim";i:2;s:4:"cory";}
    a:3
    This matches how many Network Admins you have. So, for 3 admins you'd put 3.
    i:0, i:1, i:2
    This is the user order: Admin1, Admin2, Admin3, etc.
    s:5, s:3, s4
    This matches how many characters are in the username.
    admin, tim, cory
    This is the declared username that will be associated with the Network Admins. Please note that this is case-sensitive.
  4. Based on the above syntax, the updated value will be:
    a:4:{i:0;s:5:"admin";i:1;s:3:"time";i:2;s:4:"cory";i:3;s:4:"josh"}
  5. Save the value

You now grant Josh Network Admin privileges on your WordPress Multisite Network.
